# Senai | Lógica da Programação

Este repositório reúne exemplos e exercícios feitos em sala de aula, além de duas avaliações práticas de Burger Tech. A raiz funciona como um índice visual para abrir cada item com um clique.

## Parte do Professor

Este ambiente foi organizado para apoiar aulas de Lógica da Programação no SENAI. O conteúdo da pasta mostra, na prática, a evolução dos alunos em atividades simples, exercícios guiados e avaliações mais completas.

### Organização pedagógica

- [Calculadora](calculadora/index.html): exemplo de interface web com operações matemáticas básicas.
- [Burger Kitchen](burger-kitchen/index.html): sistema de acompanhamento de pedidos, útil para trabalhar estados, fluxo e atualização de tela.
- [Aula 06](aula06.html): exercício simples de leitura de valores e soma com validação.
- [Basico](basico.html): página de estudo com elementos HTML e interação elementar.
- [Teste](teste.html): arquivo de apoio para testar lógica, mensagens e pequenos comportamentos.
- [Projeto Burger Tech](projeto_burger_tech/README.md): avaliação prática em Batch, com foco em lógica sequencial, escolhas e geração de cupom.
- [Burger Tech Web](projeto-burger-tech-web/index.html): avaliação prática em Web, mostrando a mesma ideia em HTML, CSS e JavaScript.

### O que o professor encontra na pasta

- Arquivos HTML de apoio para aula e exercícios.
- Uma calculadora simples para revisar entrada, saída e operadores.
- Um sistema de cozinha para discutir fila, status e visualização de pedidos.
- Dois projetos de avaliação prática, um em Batch e outro em Web.
- Diagramas `.drawio` e documentação em `.md` para explicar o raciocínio lógico.

## Parte do Desenvolvedor

O repositório tem como objetivo demonstrar lógica de programação aplicada a pequenos sistemas. A navegação principal fica em [index.html](index.html), que concentra os links para cada exercício e projeto.

### Visão do sistema

- [Calculadora](calculadora/index.html): interface de calculadora com HTML, CSS e JavaScript.
- [Burger Kitchen](burger-kitchen/index.html): painel de cozinha para acompanhar pedidos em preparação.
- [Projeto Burger Tech](projeto_burger_tech/README.md): aplicação em Batch com menu, escolha de lanche, combo e cupom fiscal.
- [Burger Tech Web](projeto-burger-tech-web/index.html): versão web da avaliação prática, com cálculo do pedido e exibição de resultado.
- [SA-Logica-do-sanduiche.md](SA-Logica-do-sanduiche.md): enunciado base que descreve o problema e os requisitos lógicos.

### Formas de testar

1. Abra [index.html](index.html) no navegador e siga os cartões do índice.
2. Abra diretamente cada arquivo HTML para testar os exercícios isoladamente.
3. Em [calculadora/index.html](calculadora/index.html), teste os botões, o limpar, o apagar e a operação final.
4. Em [burger-kitchen/index.html](burger-kitchen/index.html), verifique a lista de pedidos, os estados e a atualização da tela.
5. Em [projeto-burger-tech-web/index.html](projeto-burger-tech-web/index.html), teste nome, escolha do lanche, opção de combo e geração do cupom visual.
6. Em [projeto_burger_tech](projeto_burger_tech), execute o arquivo `.bat` no Windows para validar o fluxo em terminal e a criação do cupom fiscal.
7. Use os arquivos `.drawio` para conferir o fluxo lógico antes de alterar regras ou telas.
